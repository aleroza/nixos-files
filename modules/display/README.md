# display module

Display server, GPU, and desktop manager configuration.

## What it configures

- X11 base configuration
- GDM (GNOME Display Manager)
- GPU settings

## Key options

- `display.server` — display server type (x11 or wayland)
- `display.gdm` — enable GDM (bool)
- `display.gpu.enable` — GPU configuration enable (bool)
